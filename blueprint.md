
# 프로젝트 청사진

## 개요

이 웹사이트는 사용자가 방 사진을 업로드하면 인공지능을 이용해 가상으로 붙박이장을 설치한 모습을 보여줍니다. 사용자는 자신의 공간에 붙박이장이 어떻게 어울릴지 미리 확인해볼 수 있습니다.

## 기능

*   **사진 업로드:** 사용자는 자신의 기기에서 방 사진을 선택하여 웹사이트에 업로드할 수 있습니다.
*   **가상 붙박이장 생성:** 업로드된 사진에 자동으로 붙박이장이 합성된 이미지를 생성합니다.
*   **결과 비교:** 원본 사진과 붙박이장이 추가된 사진을 나란히 비교하여 볼 수 있습니다.

## 현재 작업 계획

1.  **UI/UX 디자인:** 사용자가 쉽고 직관적으로 사용할 수 있도록 현대적인 디자인의 UI를 구성합니다.
2.  **HTML 구조 설계:** `index.html` 파일에 사진 업로드 버튼, 이미지 표시 영역 등 기본 구조를 설계합니다.
3.  **CSS 스타일링:** `style.css` 파일을 수정하여 세련되고 모바일 친화적인 디자인을 적용합니다.
4.  **JavaScript 기능 구현:** `main.js` 파일에 다음 기능을 구현합니다.
    *   사진 업로드 및 미리보기 기능
    *   "붙박이장 추가" 버튼 클릭 시, 미리 준비된 붙박이장 이미지를 업로드된 사진에 합성하는 기능
    *   결과 이미지를 화면에 표시하는 기능
5.  **붙박이장 이미지 준비:** 합성에 사용할 고품질의 붙박이장 이미지를 프로젝트에 추가합니다.
